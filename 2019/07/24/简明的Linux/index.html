tags:Linux基础常识
一 Linux的文件系统结构
/ 虚拟目录的根目录，通常不会在这里存储文件
/bin 二进制目录，存放许多用户级的GNU工具
/boot 启动目录，存放启动文件
/dev 设备目录，Linux在这里创建设备节点
/etc 系统配置文件目录
/home 主目录，Linux在这里创建用户目录
/lib 库目录，存放系统和应用程序的库文件
/media媒体目录，可移动设备的常用挂载点
/mnt 挂载目录，另一挂载点目录
/opt 可选目录，常放置第三方软件包和数据文件
/proc 进程目录 存放现有硬件及当前进程的相关信息
/root
/sbin 系统二进制文件，管理员级工具
/run 运行目录 存放系统运行时文件 数据
/srv 服务目录 本地服务的相关文件
/sys 存放系统硬件信息
/tmp 临时目录，创建删除临时文件
/usr 用户二进制文件
/var 可变目录，如日志

二 Linux简单的命令
ls 【选项}】【参数】列举当前目录文件
-a 所有文件
-l 详细列出文件
-h 以人的习惯显示文件大小
-R 列出当前目录下包含的子目录中的文件
查看文件的inode编号
通配符 零个或多个字符
？一个字符
【a-i】 a到i中的字符
【！a】非a的字符
cd 【参数】切换目录
.当前目录
.. 上级目录
touch 创建文件 更新文件创建时间
mkdir【-p】递归创建目录
cp【-r】 源文件 复文件
mv移动文件 重命名
rm 删除文件
-i删除时询问
-f直接强制删除
-r 进入目录
rmdir 删除目录
file 参数 查看文件类型
查看文件内容 cat more less
cat -n给所有行加上行号
-b只给文本加行号
-T去除文中所有制表符
more less 可分页查看
tail 查看部分文件
-n 2 文件 查看最后两行
head -5 文件 查看前五行
默认情况下 tail head 10行
三监测命令
探查进程
ps 只显示会运行在当前控制台下的属于当前用户的进程
-ef查看系统上所有进程
top 实时监测
结束进程 kill killall
kill 进程ID
killall 命令
磁盘空间 df du mount
mount 挂载设备
mount -t type device directory
type：vfat Windows长文件系统
ntfs wnt，xp，w7广泛使用的高级文件系统
iso9660 cd-rom
-v 详细模式
df显示挂载磁盘使用情况
du显示特定目录的磁盘使用情况
搜索数据 grep 【options】 pattern 【file】
-v 反向搜索
-n 匹配行前加行号
-c 显示多少行匹配
压缩打包 tar -zcvf **.tar f1 f2 f3 f4
解开 tar -xvf ****.tar
